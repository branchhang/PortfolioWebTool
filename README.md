# PortfolioTool 理财账本

一个轻量的本地理财账本 PWA，用来汇总多账户资产、盈亏与走势。数据默认保存在浏览器本地，支持行情与汇率自动更新，并可导出备份文件。

## 功能简介

- 资产总览：总资产、当日盈亏、累计盈亏、收益率、曲线与结构分布
- 账户与持仓：按账户管理持仓，支持多账户筛选
- 行情与汇率：通过本地代理调用公开 API 拉取最新价格与汇率
- PWA：可添加到 iPhone 主屏幕，离线浏览已缓存页面
- 备份与恢复：导出/导入本地数据 JSON

## 使用说明

1. 需要 Node.js 18+（内置 `fetch`）。
2. 启动本地服务：

```bash
node server.js
```

3. 在浏览器打开：

```
http://localhost:8787
```

4. iPhone 访问（同一局域网）：

```
http://<你的电脑IP>:8787
```

5. 添加到主屏幕：使用 iOS Safari 打开页面 → 分享 → 添加到主屏幕。

## 数据与备份

- 数据存储在浏览器本地（LocalStorage）。
- “备份到 iCloud Drive”按钮会下载一份 JSON 备份文件，在 iOS 上可选择保存到 iCloud Drive。

## 备注

- 行情与汇率依赖公网 API；离线时会使用已缓存数据。
- 若需在手机上持续使用行情更新，请确保手机能访问你的本地代理地址。
